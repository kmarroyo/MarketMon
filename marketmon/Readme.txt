Market Monitor
Displays "real time" stock quotes.

Usually about 30 seconds behind real real time. 
Type in stock symbol and press enter. You can also send a UDP packet 
from another PC or phone using IP address and port number 60066.

If network goes down, the background will go purple and then go back to 
dark when link is back up.  

Code written in python 3 on Raspberry PI 3 B+

2-1-2019

Issues: 
Market Monitor Window may start to grow past physical size and not stop 
until machine boots.  Currently still looking at this issue.
If it happens, go to task bar and close Market Monitor window
